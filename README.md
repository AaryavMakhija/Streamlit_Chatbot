OVERVIEW - 

This project is an AI-powered chatbot built using Google's Gemini API and Streamlit. Users can interact with the chatbot through a simple web interface and receive AI-generated responses in real time.


FEATURES - 

* Interactive chatbot interface
* Powered by Google Gemini 2.5 Flash
* Conversation history support
* Secure API key management using Streamlit Secrets
* Web-based deployment with Streamlit


TECHNOLOGIES USED - 

* Python
* Streamlit
* Google Gemini API
* python-dotenv
